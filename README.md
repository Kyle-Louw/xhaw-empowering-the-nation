A responsive web application for Empowering the Nation, a skills training initiative based in Johannesburg. The platform allows users to browse vocational courses, calculate fees with discounts, and submit contact requests.

## Project Overview

Module: XHAW5112

The design is built around the Ubuntu philosophy and targets domestic workers, gardeners, nannies, and employers looking to invest in skills development.

## Features

- Home page with course navigation
- Six-month course pages: First Aid, Sewing, Landscaping, Life Skills
- Six-week course pages: Child Minding, Cooking, Garden Maintenance
- Fee calculator with automatic discount logic and VAT
- Contact page with venue selection and map integration
- Form validation and error handling
- Fully responsive layout for web and mobile

## Tech Stack

- HTML5, CSS3, JavaScript
- Font Awesome 6 for icons
- Google Maps (map library)  embed for venue locations
- Poppins font via Google Fonts

## Colour Palette

Terracotta Clay #E2725B is used for buttons, headers, and headings. Forest Green #228B22 is used for fees, active states, and accents. Golden Harvest #DAA520 is used for icons and logo details. Soft Cream #FFF8E7 is the background colour. Charcoal Grey #333333 is used for body text.

## Discount Logic

One course selected receives no discount. Two courses receive a 5% discount. Three courses receive a 10% discount. Four or more courses receive a 15% discount. VAT of 15% is applied after the discount.

## Setup

1. Clone the repository

```
git clone https://github.com/yourusername/empowering-the-nation.git
```

2. Navigate into the project folder

```
cd empowering-the-nation
```

3. Open the web app in your browser

```
open index.html
```

No build tools or dependencies are required. The project runs entirely in the browser.
